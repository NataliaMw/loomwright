iv. six specialists, two frameworks, one room

ALoopArchitect
Designs a task-specific verification loop and defends it.
CLoopCritic
Attacks the exit condition; recruits missing critics.
RLoopRunner
Starts the loop, finalizes once the exit holds.
ACodeAuthor
Writes each revision; never reviews its own work.
QQAEngineer
Really executes the checks against the candidate.
RRivalReviewer
Adversarial rival model; bounces failing revisions.
Real — all six registered & public on Band · Pydantic AI + LangGraph

ix. the real bug · the real fix · the real failure

revision 0 — buggydef page(items, page_num, size): start = page_num * size end = start + size - 1 # drops last row return items[start:end]
revision 1 — shippeddef page(items, page_num, size): start = page_num * size end = start + size # slice end is exclusive return items[start:end]
repro-test ❌ AssertionError: page returned a short page assert len(page(list(range(10)), 0, 3)) == 3

Can't make your tests pass in budget? It stops and hands you a verified failure — not a confident wrong answer. The failure mode of agentic coding, closed.
