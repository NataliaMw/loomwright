Loomwright

Loop engineering is the 2026 skill: the unit of agentic coding isn't the prompt, it's the loop — generate, check, revise, until a real exit condition holds. But everyone hand-builds those loops, and every coding-agent tool runs the same fixed pipeline for every task. Loomwright is a Band room that designs a loop for the specific task, then runs it — and recruits the specialists that task needs, on demand.
