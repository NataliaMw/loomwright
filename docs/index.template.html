🧵 Loomwright

the room that engineers the loop, then runs it

Loop engineering is the 2026 skill: the unit of agentic coding isn't the prompt, it's the loop — generate, check, revise, until a real exit condition holds. Everyone hand-builds those loops, and every coding-agent tool runs the same fixed pipeline for every task. Loomwright is a Band room that designs a loop for the specific task and runs it — and the checks below are real: they execute the generated Python in a sandbox and pass/fail on the result.
